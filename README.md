## about

A simple React app that lists Digimon from an [API](https://digimon-api.vercel.app/), includes name search, and visually highlights Mega-level characters.

## getting started

To run the project locally, make sure you have [Node.js](https://nodejs.org/en/) and a package manager such as [npm](https://docs.npmjs.com/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/laianesuzart/digimon.git
   ```
2. Navigate to the project folder:
   ```bash
   cd digimon
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
